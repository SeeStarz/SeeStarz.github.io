My Projects

Sovereign Chess

This is a desktop game made in C++ with SFML library for the chess variant "Sovereign Chess" by Mark Bates. My biggest and proudest project so far. All of the logic are written by me in C++, complete with it's own button event system. The only library used is SFML to deal with graphics, audio, windowing, user input, and TCP requests. This project taught me a lot about C++ and it's dangling pointers. Once i was stuck for hours debugging just to realize that extending a vector changes it's location in memory, rendering all pointers to it's members dangling. This project also taught me about C++'s build system. And yes, it is a big mess. I used vscode's manual compile script at the beginning and later created my own script for MinGW Make and GnuMake. By adding linux ports I also have a bit more awareness about the different build systems there is. And the fact that different compilers really don't want to agree on some things sometimes.

Minecraft Mods

I published two minecraft mods. Strong Netherite Bow Mod, a minecraft mod adding a new bow made of netherite that is stronger, but also unique by prolonging it's charge duration. Originally meant for elytra fights midair. All stats are configurable via a config file. The bow mod is then complemented by Fast Projectile Fix Mod to fix rendering issue. Both mods are developed for Minecraft forge version 1.16.5. How the mod is created is pretty straight forward, by following some youtube tutorial and mimicking some of Minecraft's original code, it is possible to add a completely new bow that's compatible with the original. There are some areas of code that doesn't make sense why it is, but that is how it works. There are also some areas where it's needed to hook with Minecraft's event handling system, notably the FOV change. Lastly any projectile above a certain speed will render absolutely incorrect, despite having a correct trajectory and destination. This happened due to Minecraft clamping values between -4 and 4. To fix this Fast Projectile Fix was deployed separately to override these mechanics and instead send a proper double type value.
